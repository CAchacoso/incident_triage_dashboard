# Incident & Triage Analytics Dashboard

**Tools:** Python, Streamlit, Plotly, Pandas  
**Project Status:** In Progress  

## Overview

Interactive dashboard built with Python, Streamlit, Pandas, and Plotly.

## Features
- Incident KPIs (Open, Closed, MTTR)
- Severity and trend analysis
- Service-level breakdowns
- Aging incident detection
- Interactive sidebar filters

## Getting Started 
1. **Clone the repository**
```bash
git clone https://github.com/CAchacoso/bug-test-dashboard.git
cd bug-test-dashboard
```
2. **Install dependencies**
```bash
pip install -r requirements.txt
```
3. **Run the program**
```bash
streamlit run app.py
```
